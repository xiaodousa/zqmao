    2016-8-24 11:11:11 by毛神

    华米科技打造的小米手环大卖，为什么手表又要和小米撇清关系呢？行业分析师潘九堂对此给出了一个看起来颇有道理的解释。
    有人认为，华米这款智能表不打小米品牌，是因为品质、性能、价格达不到小米的要求，或者华米和小米有矛盾。
    但是潘九堂猜测，或许是因为华米想独立A股上市，以减少对小米依赖和关联交易。
    他还透露，这是华米独立打造的首个爆款产品，会在中国和美国两地发售，不用担心品质、性能、价格，或许还有惊喜。
